# Walmart-Sales-Analysis
"Data Analytics internship assignment – Walmart Sales Dashboard"
Power BI Dashboard Project

## 📌 Project Overview
This project demonstrates the use of Microsoft Power BI to transform raw financial data into meaningful business insights. It includes interactive dashboards, data modeling, DAX calculations, drill-through reports, conditional formatting, and financial performance analysis.

## 🎯 Project Objectives
- Connect and transform financial data.
- Build an efficient data model.
- Create DAX measures for business calculations.
- Develop interactive dashboards and reports.
- Analyze sales, profit, and business performance.
- Enable data-driven decision-making through visual analytics.

## 🛠️ Tools & Technologies
- Microsoft Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query Editor
- Data Modeling
- Microsoft Excel (Dataset)
